# Image analysis agent training

### Files

- `20260522_bag380_raw.jpeg`: Raw image without annotations
- `20260522_bag380_annotated_imageJ.jpeg`: Image with annotations in ImageJ format that needs to be viewed in ImageJ/Fiji
-  `20260522_bag380_annotated_flattened.jpeg`: Image with flattened annotations that can be viewed as a regular image
- `20260522_bag380_data.xlsx`: Size data for length and width measurements of each oyster in annotated images 

### Goal

- Get an AI agent to be able to identify oysters and measure length and width
